# Run Foxy Run!
Runner game based on Godot game engine and C# language.
The purpose of the program to learn and practice godot and to make some satisfaction on finishing stuff.

![Game Screenshot](doc/readme_screen_shot.png)

## Stack
* Godot v3.4.4
* C#/Mono
